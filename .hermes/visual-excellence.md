# Visual Excellence — Standards Awwwards Level

> Standard : chaque site Hermes doit être digne d'une nomination Awwwards.
> Ce document définit les règles de motion, typographie créative, layouts éditoriaux et micro-interactions.
>
> **Règle de compatibilité** : toutes les animations respectent `prefers-reduced-motion`.
> **Règle de performance** : chaque effet doit être GPU-accelerated (transform + opacity uniquement).

---

## État du projet clement-geneau

- [x] `prefers-reduced-motion` géré dans `globals.css`
- [x] Micro-interactions CSS sur boutons et liens (hover states)
- [x] Typographie contrastée : Fraunces display + Inter body
- [x] Couleurs OKLCH cohérentes sur toute la palette
- [ ] Smooth scroll (Lenis) — non implémenté (optionnel pour ce projet)
- [ ] Animations GSAP/Framer — non implémentées (optionnel)
- [ ] Curseur custom — non implémenté (optionnel)
